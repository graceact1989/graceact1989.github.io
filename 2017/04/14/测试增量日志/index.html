定时任务遇到的问题

  • win10任务计划服务不可用

    解决方案
查看remote proceducre Call(RPC)和windows event log 是否关闭.如果关闭则开启.
如果开启则 在开始---运行中输入cmd，打开后输入netsh winsock reset后确定，然后重启计算机即可解决。
需要使用管理员权限启动cmd
